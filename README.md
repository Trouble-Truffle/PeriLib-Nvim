# PeriLib

A bunch of functions, most of the Functional Programming variety.

Apart from most Non-IO functions, this library takes use of Neovim functions as such is mainly useless outside Neovim

  - `PeriLib.Data.Monoid`
  - `PeriLib.Data.Tuple`
